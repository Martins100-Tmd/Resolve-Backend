# usage

1. ## Clone repo

```bash
git clone https://github.com/Martins100-Tmd/Resolve-Backend.git
cd resolve
```

2. ## install packages

```bash
npm installl
```

3. ## Create you `.env` file in the root folder and add 

```bash 
MONGO_DB_URI=add_your_mongodb_uri
```


4. ## start program 

```bash 
npm run start
```
